GATA2 Variant Explorer

How to use the explorer

  • Use the search box (top right) to search across all columns.
  • Use the column filters (under each header) to filter specific fields.
  • Helpful search terms to try:
    • Variant type: missense, nonsense, frameshift, splice_site, inframe_indel, inframe_ins, inframe_del
    • Protein region: N_tail, N_finger, spacer, C_finger, C_tail
    • Origin: germline, somatic, mixed
    • Database IDs: ClinVar, All_of_Us, gnomAD, COSMIC
    • HGVS: c., p., or a position like 345 or 354
  • Tip: You can combine terms in the global search (e.g., C_finger missense).
